    1.1 디지털과 아날로그
        아날로그 신호 : 자연계에서 일어나는 물리적인 양은 시간에 따라 연속적으로 변화
                      온도, 습도, 소리, 빛 등은 시간에 따라 연속적인 값을 갖는다.
        디지털 신호   : 분명히 구별되는 두 레벨의 신호값만을 갖는다.
*두 신호의 차이점은 연속적인가 비연속적인가*

    1.2 디지털 시스템과 아날로그 시스템
        디지털 시스템 : 이산적인 정보 가공, 처리하여 최종 목적으로 하는 정보를 출력하는 모든 형태의 장치
        아날로그 시스템 : 연속적인 정보를 입력받아 처리해서 연속적인 정보를 출력하는 모든 형태의 장치

    1.3 디지털 시스템의 장점
        * 디지털 시스템의 많은 장점으로 인하여 기존 아날로그 시스템이나 새로운 시스템의 대부분은 디지털 시스템으로 구성*4

    2.1 디지털 정보의 전압 레벨
        디지털 정보를 표현하기 위해 2진수 체계를 사용
        0과 1만의 2종류 디지트(digit)를 사용
    
    2.2 디지털 정보 표현 단위
        1nibble = 4bit
        1byte = 8bit
        1byte = 1character
        영어로는 1byte로 1 문자 표현, 한글은 2 byte 필요
        1word = 64bit
        1word : 특정 cpu에서 취급하는 명령어나 데이터의 길이에 해당하는 비트 수 

        MSB : 최상위 비트는 (MSB 가장 왼쪽)
        LSB : 최하위 비트는 (LSB 가장 오른쪽)


        SI 단위와 IEC단위 비교
        tera peta exa zetta 
        제엑페테 2^21 2^18 2^15 2^12

    2.3 정보를 표현하기 위한 비트 수
        비트 n 개로는 2^n가지 정보를 표현할 수 있다. 
        m가지 정보를 표현하려면 비트 ceil(log2 m) 개가 필요하다. 

    3.1 펄스 파형
        펄스 파형은 low상태와 high 상태를 반복하는 전압레벨로 구성
        주기 펄스 & 비주기 펄스로 분류
        
        펄스 폭(pulse width ) 중요

        주파수 : 주기적인 파형이 1초동안에 진동한 횟수를 의미
                단위는 전파를 처음으로 발견한 독일의 물리학자 헤르츠의 이름을 따서 헤르츠 사용
        주기 : 주기적인 파형이 1회 반복하는데 걸리는 시간을 의미

        주파수가 1khz일때 주기는 1/1000

        주파수와 주기의 관계 : 반비례함
        듀티 사이클 : 주기 T에 대한 펄스 폭의 비를 백분율로 정의
        듀티 사이클을 충격계수라고도 한다

        예제 1-4 풀이에서 나오는 us에서 u는 마이크로 us는 마이크로 세컨드
        M(마이크로) = 10^-6
        m(밀리) = 10^-3

    4.1 디지털 집적회로
        디지털 회로: 디지털 정보를 처리하는 디지털 시스템의 하드웨어를 말한다
        디지털 하드웨어, 디지털 회로, 논리회로 등을 특별히 구별하지 않고 사용한다
        집적회로 : 작은 실리콘 칩 위에 저항, 커패시터, 다디오드, 트랜지스터 등의 전자부품을 여러 단계의 공정을 거쳐서
                  내부적으로 상호 연결한 것을 의미한다
        칩: 실리콘 반도체를 의미한다.

        IC 패키지
        PCB에 장착하는 방법에 따라 삽입장착형과 표면실장형으로 구분
        삽입 장착형 IC는 PCB 보드의 구멍에 끼우는 핀을 가지고 있어, 뒷면의 도체에 납땜으로 연결할 수 있으며,
        DIP 형태를 갖는다.
        표면 실장형 IC는 PCB표면의 금속 처리된 곳에 직접 납땜처리
        SMD는 DIP형태의 논리회로 크기를 70%줄이고 무게를 90%감소
        SMD는 PCB의 제조가격을 크게 하락시킴

        집적회로의 분류
        소자 수에 따른 집적회로의 분류

        소규모 집적 회로 / SSI / small scale IC
        중규모 집적회로 / MSI / medium scale IC
        대규모 집적 회로 / LSI / large scale IC
        초대규모 집적 회로 / VLSI / very large scale IC
        극초대규모 집적 회로/ ULSI / ultra large scale IC
    
    5.1 아날로그-디지털 변환과정
        아날로그 신호 - 표본화 - 양자화 - 부호화 - 디지털 신호

        표본화 : 샤논의 표본화 정리 : 신호의 최고 주파수의 2배 이상의 빈도로 샘플링하면 샘플링된 데이터로부터 본래 데이터를 재현 가능
        사람 음성인 경우 1초동안에 8000번 샘플링 필요

        양자화 : 펄스의 진폭의 크기를 디지털 양으로 변화하는 것
                이 과정에서 불가피한 양자화 잡음이 발생
                양자화 잡음은 미리 정한 신호 레벨의 수를 늘리거나 줄일 수 있으나, 데이터의 양이 많아지는 단점이 있다

        부호화 : 펄스의 전폭의 크기를 부호화하는 것
