1. 인공지능과 함께하는 우리 사회
    
    1.1 가상세계, 메타버스
        
    메타버스의 개념 : 증강과 시뮬레이션, 내적인것과 외적인 것인 두 측을 기준으로 4가지 유형 분류
                    - ASF(기술 연구단체)에서 분류함

    증강현실 : 현실 공강에 2D OR 3d로 펴현한 가상의 정보를 표시해 상호작용하는 환경
    
    일상기록 : 블로그처럼 자신의 인생 기록
    
    거울세계 : 구글 어스 처럼 디지털 정보를 기반으로 실제 세계를 사실적으로 반영한 섹{}
    
    가상세계 : 현실 세계의 경제적, 사회적인 활동과 유사한 활동을 할 수 있도록 디지털 데이터로 구축한 세계

    메타버스 활용 : 회사, 대학, 가수, 정치인, 게임, 컨퍼런스, 교육, 엔터, 음악, 콘텐츠, 산업 현장 등


    1.2 가상현실 : 현실세계가 아닌 가상세계를 체험할 수 있는 기술
        - 인공적으로 특정 상황, 환경에 처한 것처럼 느끼도록 하는 기술
        - 오감 체험으로 발전 : 기본적으로 시청각 미각 후각 촉각까지 범위 확대 / 입출력 장치, 컴퓨터 그래픽 기술 필수적 / 영상출력 장치는 HMD가 대표적

        증강현실 : 현실세계를 그대로 경험하면서, 부가적으로 가상의 디지털 정보를 덧붙여 현실세계를 강화하는 기술
        - 증강현실(AR) :  현실세계의 영상에 가상의 물체나 필요 정보를 합성 / HMD같은 장비 없이 스마트폰만으로 체험할 수 있기에 가상현실보다 대중화
        - 사례 : 포켓몬고, 실제적 관찰과 이해를 돕는 증강현실 기술 / 게임, 패션, 가구 구조, 네비게이션, 의료, 교육 / 최근에는 제조업과 의료 분야

        혼합현실 : 가상현실과 증강현실의 단점을 보완해 더욱 진화된 가상세계를 구현하는 기술
        - 현장 작업자 또는 사용자 친화적으로 상호작용할 수 있는 효과적인 방법으로 활용 

        확장현실 : 가상, 증강, 혼합 현실 기술을 모두 지칭하는 용어
        - 위 3개 현실 기술의 기반 / 가상과 현실을 융합해 현실의 경험을 확장하고 몰입감 제공 / HMD 기기 활용


    
2. 인공지능과 삶의 변화
    Tensor : 차원에 따른 데이터의 명칭
             - 데이터가 몇 차원으로 구성되어 있느냐에 따라 우리가 흔히 쓰는 용어들이 결정
             0차원 텐서 : 스칼라 / 하나의 숫자값
             1차원 텐서 : 벡터 / 숫자들이 일렬로 나열된 형태
             2차원 텐서 : 행렬 / 숫자들이 가로와 세로로 나열된 것.
             3차원 텐서 : 텐서 / 행렬을 여러 겹 쌓아올린 입체적인 구조. 3차원부터는 그냥 텐서로 통칭해서 부름
