






*키오스크와 POS*
1. 키오스크(kiosk) 
    - 터치패널 등이 달린 작은 설치형 디지털 단말기
*사용자 메뉴를 고르고 결제를 하는 상호작용 키오스크로 많이 사용*
*입력장치에서 중앙처리와 출력 장치까지 탑재된 또 다른 컴퓨터중 하나*
2. POS(Point of Sale) 시스템
    - 다양한 매장에서 사용하는 결제 시스템(ex. T머니 장치)


*CPU*
1. CPU
    - 입력된 자료를 적절한 정보로 변환하기 위하여 소프트웨어로부터 받은 명령어를 실행하는 장치
    Central Processing Unit
    - Microprocessor 줄여서 Processor 라고도 부름
    - 제어장치 (control unit)
        자료와 명령어의 중앙처리장치로의 입출력을 제어
    - 연산장치(arithmetic logic unit)로 구성
        산술연산과 논리연산을 수행하는 장치
    - 주요 생산 회사
        인텔, 암드 


*메인보드*
1. 메인보드
    - 중앙처리장치는 마더보드라고도 부르는 컴퓨터 내부의 메인보드에 장착
    주기판 혹은 모기판
    - 중앙처리장치를 연결하는 소켓
    - 주기억장치를 설치하는 메모리 슬롯
    

*그래픽 처리장치(Graphics Processing unit)*
    - 1999 엔비디아 gpu 지칭
    - gpu는 고속의 병렬처리로 대량의 행렬과 벡터를 다루는데 뛰어난 성능을 발휘
    - GPU 활용 : 빅테더이 처리, 인공지능의 심층신경망 분야

*모바일AP*
    스마트폰, 태블릿 등의 모바일 기기에 사용되는 cpu를 Application Processor
    
1. 모바일 ap
    - 모바일 기기의 두뇌 역할을 담당하는 반도체 칩
    - 연산을 위한 cpu의 주된 기능 외 
    - 여러가지 시스템 장치와 인터페이스를 제어 기능
    - 메모리와 표시장치를 위한 그래픽처리, 카메라, 저장장치, 무선통신
    - 디지털 신호 처리, 이미지 처리 등
    - 엄지손톱보다 작음
    - 여러 기능의 반도체를 한 개의 칩에 모아 만든 시스템온 칩

2. 대표적인 ap
    - 웰컴의 msm시리즈 웰컴
    - 스냅드래곤
    - 삼성 엑시노스
    - 엔비디아 테그라 등

*주기억장치*
    cpu에서 처리중인 프로그램과 데이터를 임시로 저장하기 위한 장치
1. 임의접근 메모리 RAM(Random Access Memory)
    - 메모리의 임의의 위치에 데이터를 읽거나 쓸 수 있음을 위미
    - 휘발성 메모리
    - 듀얼 인라인 메모리 모듈(DIMM)
    - 여러 개의 메모리 칩을 회로기판 위에 탑재한 메모리 모듈
    - 마더보드의 dimm 소켓이라 부르는 자치에 장착
2. 읽기전용 메모리 ROM(Read only Memory)
    - 비휘발성 메모리
    - ram과 반대되는 개념, 현대 컴퓨팅에서는 플래시 메모리 기술로 통합되어가고 있다.
    - 하드웨어 제어 프로글매은 지워지지 않는 메모리에 저장된다.(펌웨어 BIOS)


*보조 기억장치*
- 프로그램이나 데이터를 저장하기 위한 저장 공간으로 간단히 저장장치라고도 부름
1. 저장장치
    - 주기억장치에 비하여 속도는 느리지만, 가격이 저렴하며 더 넓은 공간으 ㄹ이용

2. 종류
    - USB
    - HDD
    - SSD
    - external HDD
