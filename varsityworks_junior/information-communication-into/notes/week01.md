1장. 정보통신의 개요

1_A. 정보통신의 정의

    데이터(Data)
    - 인간이나 어떤 자ㅏ동화 도구를 이용하여 현실세계로부터 단순한 관찰이나 측정을 통하여 수집한 사실, 개념, 명령들에 대한 값을 의미
    정보(information)
    cf. text의 의미 또한 가지고 있음
    - 데이터를 처리 가공하여 의미를 부여한 결과
    통신(Communication)
    - 송수신자가 전송 매체를 통해서 정해진 규칙을 사용하여 의미를 전달하는 것을 의미

    cf. 통신에서의 규칙: 프로토콜(protocol)

    정보통신
    - 정보의 이동행위로써, 정보를 서로 공유함으로써, 정보의 가치를 향상시키는 것을 의미

    1_A_a.데이터의 종류
    음성(Voice) 
    - 가장 많이 사용하고 있는 정보형태, 교환 서비스, 음성 원격 회의, 음성 정보 서비스 등에 이용
    데이터(Data)    
    - 음성을 제외한 모든 다른 정보를 의미, 전자우편, 파일 전송 서비스 등에 이용
    화상(Image)
    - 그림이나 그래픽 등의 정보, 팩시밀리, 그래픽 전송 서비스등에 이용
    영상(Video)
    - 화상 통신의 진보된 형태, TV방송, 영상회의, 비디오 텍스, 영상 응답 시스템, 원격 회의
    멀티미디어(Multimedia)
    - 모든 정보들을 통합하고 있는 형태를 의미, 원격 교육, 원격 회의 서비스등에 이용
 
1_B. 통신 시스템의 구성

    데이터 처리게
    - 컴퓨터 : 중앙 처리 장치, 주변 장치로 구성
    데이터 전송계
    - 단말장치
    - 데이터 전송 장치 : 신호 변환 장치, 통신 회선
    - 컴퓨터

    데이터 통신장치(DCE)
    - 신호 변환 기능: 변복조기(모뎀), 디지털 서비스 장비( DSU: Digital Service Unit)
                    전송 신호의 동기 제어 기능
                    송수신 확인 기능
                    전송 오류의 검출 및 정정 기능

    OSI 참조 모델(Reference Model)
        - 서로 다른 시스템 간의 통신을 위한 표준안을 제공함으로써
          통신에 방해가 되는 기술적인 문제점을 제거하였으며, 시스템간의 정보 교환을 하기 위한 상호 인터페이스를 정의하였다.
